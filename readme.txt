========================================================================
                           RC-Car Project
========================================================================


DESCRIPTION
-----------
The code provided is for a remote-controlled robotic car which I had developed over my break, which uses 4 x DC motors, a motor shield and an IR sensor to control the direction and movement.


INSTALLATION & SETUP
--------------------
1. Ensure that you have an IDE such as the Arduino IDE installed and a C++ compiler (which is typically pre-installed with the Arduino IDE)
2. Install the source code.
3. Install the following modules/ external libraries (if not already installed) and import them:
	- AFMotor.h 	
	- IRemote.hpp
4. Upload the code to the Arduino.
5. Whilst keeping the cord plugged in, configure the IR signals for your remote using the serial plotter if not already configured.
